# 'Guess the Number game'
## Small University Project

## Installation 
Using git command:
```
git clone https://github.com/Corvothanov/number-game.git
```
## Running the game
Simply run the game.exe file you will be met with option to start the game or check the scoreboard (if possible).

## Features
- main menu with option to start the game or check the scoreboard divided by difficulty,
- difficulty options,
- random messenger upon wrong guess,
- tries counter,
- saveable to file score after finishing the game
